# 作品集

## 作品一：三子棋人机对弈系统

嵌入式负责人（硬件设计+运动控制+博弈算法）
时间：2024.07-2025.08
情境：为实现一套能够自主完成棋盘识别、棋子抓取和智能对弈的三子棋机器人系统，需要解决视觉识别、运动控制精度以及实时决策等问题。
任务：负责系统整体嵌入式方案设计，包括硬件电路设计、运动控制算法开发、视觉通信接口设计及AI博弈算法实现。

行动：

1. 使用嘉立创EDA完成STM32F103主控板及步进电机驱动扩展板设计，引入继电器控制电磁铁取机构，提高系统可靠性；
2. 采用编码器反馈结合PID闭环控制算法，实现双步进电机高精度运动控制；
3. 通过UART与OpenMV视觉模块通信，实现棋盘与棋子坐标实时获取；
4. 基于Minimax算法进行优化，引入快速查表机制提升决策效率；

结果：

1. 实现棋子自动识别、抓取和落子全流程自动化；
2. 定位精度达到 ±0.5mm；
3. AI决策时间缩短至 100ms以内；
4. 系统运行稳定，无阻塞死机现象，具备良好的人机交互体验。

## 作品二：RoboMaster哨兵机器人

嵌入式软件负责人
时间：     2024.10-2026.04
情境：作为全国大学生机器人竞赛项目，需要构建具备自主决策能力的哨兵机器人，实现云台、底盘和发射机构的高实时协同控制。
任务：负责机器人嵌入式控制系统开发，包括软件架构设计、RTOS移植、电机控制网络搭建以及姿态解算算法实现。
行动：
1. 设计并搭建 BSP-Module-App 三层软件架构，提高代码复用性与可维护性；
2. 移植FreeRTOS，划分云台控制、底盘运动和发射管理任务，通过优先级调度和信号量同步保证实时性
3. 构建CAN总线电机控制网络，并设计ID仲裁机制避免总线冲突；
4. 基于BMI088传感器开发四元数姿态解算算法，结合卡尔曼滤波提高测量精度；
5. 实现带梯形积分、低通滤波及输出限幅的串级PID控制器，完成位置-速度-电流三环控制；
6. 设计串口通信协议，实现双MCU间高速数据交互。
7. 人机交互PWM实现上电呼吸灯指示系统状态。

结果：

1. 云台控制响应时间控制在 5ms以内；
2. 关键控制任务响应延迟低于 1ms；
3. 姿态解算精度达到 ±0.1°；
4. 代码可维护性与复用性显著提升，便于后续迭代开发。
5. 项目获得 2025 RoboMaster联盟赛国家二等奖和2026 RoboMaster联盟赛国家二等奖。

## 作品三：电机控制驱动器

![电机控制驱动器](../portfolio/motor_driver.jpg)

FOC 无刷直流电机驱动器，用于机器人关节控制。

- **硬件：** STM32G431 · 三相逆变桥 · 霍尔/编码器接口
- **固件：** 双环 PID（速度环 + 电流环）· SVPWM
- **性能：** 转速闭环精度 ±1 RPM · 20kHz 开关频率
